## Study
### 1. flex-wrap
flex-item 요소들이 강제로 한줄에 배치되게 할 것인지, 또는 가능한 영역 내에서 벗언지 않고 여러행으로 나누어 표현할 것인지를 결정하는 속성

|속성값|의미|
|---|---|
|nowrap|기본값, 부모요소 영역을 벗어나더라도 flex-item요소들을 한줄로 배치|
|wrap|여러 행에 걸쳐서 배치|
|wrap-reverse|wrap 속성값과 동일하지만, 요소가 나열되는 시작점돠 끝점의 기준이 반대로 배치|
