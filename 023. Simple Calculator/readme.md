## 학습 
### switch 구문 
복수의 if 조건문은 switch문으로 바꿀 수 있습니다.    

switch문은 하나 이상의 case문으로 구성됩니다.
```
switch(operator) {
    case "plus" : 
      result.innerHTML = Number(num1) + Number(num2);
      break;

    case "minu" : 
      result.innerHTML = Number(num1) - Number(num2);
      break;
```
